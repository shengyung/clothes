（人像照 + 服裝照） S3-->>API: presigned URLs API->>Fashn: 呼叫 tryon-v1.6
傳入 presigned URLs Fashn->>S3: 直接拉取圖片 Fashn-->>API: { fashn_task_id } loop 輪詢 Fashn.ai 狀態 API->>Fashn: GET 任務狀態 Fashn-->>API: status: 'processing' end Fashn-->>API: status: 'completed'
result_image_url API->>Fashn: 下載結果圖片 API->>S3: 上傳結果圖至 S3 S3-->>API: 儲存成功 API->>DB: 更新 TryonTask

📊 TryonTask 狀態機

試穿任務的生命週期狀態轉換

stateDiagram-v2 [*] --> pending : POST /api/tryon 建立任務 pending --> processing : BackgroundTask 開始執行 processing --> completed : Fashn.ai 推論成功
結果圖上傳 S3 processing --> failed : API 呼叫失敗
或推論逾時 completed --> [*] failed --> [*] note right of pending 前端開始輪詢 每 3 秒一次 end note note right of completed result_image_url 填入 前端顯示結果 end note note right of failed error 欄位記錄原因 前端顯示錯誤訊息 end note
